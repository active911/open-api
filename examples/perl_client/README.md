### Perl Client

This perl client will connect to the Active911 open API service and perform simple commands.

Before running the client, make sure you have the following Perl modules installed:

- JSON
- Data::Dumper
- LWP::UserAgent

#### Example usage

(replace 'xxxxxx' and 'yyyyyy' with your app key and API key, respectively):

```bash

# client.pl xxxxxx yyyyyy ping

```
